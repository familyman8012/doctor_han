## 11. 권장 구현 로드맵

```
[Phase 0: 준비] ─────────────────────────────────
  ├─ TossPayments 가맹점 등록
  ├─ 파일럿 업체 3~5개 확보
  ├─ 가격 정책 확정 (인터뷰)
  └─ 약관 작성

[Phase 1: 크레딧 + 결제] ────────────────────────
  ├─ 크레딧 시스템 (충전/차감/조회)
  ├─ TossPayments 연동
  └─ 업체 가격 설정 UI

[Phase 2: CPL 건별] ─────────────────────────────
  ├─ 리드 생성 시 크레딧 차감
  ├─ 중복 리드 체크/복구
  ├─ 무응답 자동 환불
  └─ 잔액 부족 알림

[Phase 3: CPL 기간제] ───────────────────────────
  ├─ 구독 상품 구매
  ├─ 구독 중 리드 무료 처리
  └─ 만료 알림

[Phase 4: 정산/환불] ────────────────────────────
  ├─ 환불 요청/처리
  ├─ 월별 정산
  └─ 리포트

[Phase 5: CPA/비딩/쇼핑몰] ──────────────────────
  ├─ (업체 협조 확보 시) CPA
  ├─ (매니저 준비 시) 비딩
  └─ (필요 시) 쇼핑몰
```
