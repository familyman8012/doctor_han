# 메디허브(Medihub) 페이지 구조

## 사용자 유형별 페이지 구조

### 1. 공통 페이지
| 페이지 | 설명 |
|--------|------|
| 랜딩/홈 | 서비스 소개, 검색 진입점 |
| 회원가입 | 일반(한의사/업체), 소셜(네이버/카카오/구글) |
| 로그인 | 일반 + 소셜 로그인 |
| 계정 찾기 | 아이디/비밀번호 찾기 |
| 이용약관/개인정보처리방침 | 법적 문서 |

---

### 2. 한의사(의료인) 페이지
| 페이지 | 핵심 기능 |
|--------|----------|
| **검색/탐색** | 카테고리 트리, 필터(가격/평점), 정렬, 키워드 검색 |
| **업체 리스트** | 썸네일, 별점, 가격, 빠른응답 뱃지 |
| **업체 상세** | 서비스 설명, 포트폴리오, 리뷰, FAQ, 문의하기 |
| **문의 작성** | 문의 양식 (서비스명, 연락처, 문의내용 등) |
| **마이페이지** | |
| ├ 대시보드 | 진행중 리드, 대기 견적, 알림 |
| ├ 내 문의함 | 문의 상태 관리 (대기/진행/완료) |
| ├ 찜 리스트 | 즐겨찾기 업체 |
| ├ 조회 목록 | 최근 본 업체 |
| ├ 리뷰 관리 | 내가 쓴 리뷰 |
| ├ 알림 설정 | 알림 수단/종류 on/off |
| └ 계정 관리 | 프로필 수정, 비밀번호 변경, 탈퇴 |

---

### 3. 업체 회원 페이지 (파트너 센터)
| 페이지 | 핵심 기능 |
|--------|----------|
| **대시보드** | 수신 리드 요약, 응답률, 전환율 |
| **프로필 관리** | 업체 정보, 서비스 소개, 포트폴리오 등록 |
| **리드 관리** | 받은 문의 목록, 상태 관리, 견적 제출, 메시지 스레드 |
| **리포트/통계** | 노출/클릭/리드 수, 기간별 비교, ROI 계산 |
| **프로모션/광고** | 우선노출 신청, 뱃지 신청 |
| **리뷰 관리** | 받은 리뷰 확인, 신고/이의제기 |
| **정산/결제** | 크레딧 충전, 결제 내역, 세금계산서 |
| **알림/메시지** | 실시간 알림, 메시지함 |

---

### 4. 관리자(Admin) 페이지
| 페이지 | 핵심 기능 |
|--------|----------|
| **대시보드** | DAU/MAU, 리드 수, 응답률, 매출, 오늘 할 일 |
| **회원 관리** | 한의사 목록, 승인/제재, 권한 관리 |
| **업체 관리** | 업체 목록, 승인/심사, 제재 이력 |
| **카테고리/태그** | CRUD, 배지/라벨 정책 |
| **광고/노출 관리** | 슬롯 관리, 캠페인, 배너 설정 |
| **리드 모니터링** | SLA 추적, 대화 스레드 열람, 자동 규칙 |
| **과금/정산** | 요금제 관리, 정산 리포트, CSV 내보내기 |
| **환불/보상** | 정책 엔진, 자동 처리, 이의제기 심사 |
| **콘텐츠/리뷰/신고** | 리뷰 품질 관리, 신고 처리, FAQ/공지 |
| **통계/분석** | 리드 퍼널, 광고 성과, 이탈 분석 |

## 질문 사항

1. **MVP 범위** - V1 기준으로 먼저 가시나요? (검색/비교, 리드 연결, 리뷰 기본)
2. **회원 유형** - 한의사 + 업체 둘 다 처음부터 구현하나요?
3. **소셜 로그인** - 네이버/카카오/구글 중 우선순위가 있나요?
4. **결제 시스템** - 초기에 정액제로 간다고 했는데, PG 연동 범위는?

  1. 검색/탐색 페이지                                                                                                                                                                             
                                                                                                                                                                                                  
  의료 B2B 업체 검색 페이지를 디자인해줘.                                                                                                                                                         
                                                                                                                                                                                                  
  상단:                                                                                                                                                                                           
  - 로고(좌측), 검색바(중앙, 넓게), 로그인/알림 아이콘(우측)                                                                                                                                      
  - 검색바는 돋보기 아이콘과 "업체명, 서비스, 키워드 검색" 플레이스홀더                                                                                                                           
                                                                                                                                                                                                  
  카테고리 영역:                                                                                                                                                                                  
  - 대분류 카테고리를 아이콘+텍스트 형태의 카드로 가로 나열 (원외탕전, 의료기기, 인테리어, 간판, 전자차트, 마케팅, 세무/노무, 홈페이지)                                                           
  - 각 카드는 호버 시 살짝 올라오는 효과                                                                                                                                                          
  - 선택하면 하위 카테고리가 아코디언 또는 드롭다운으로 펼쳐짐                                                                                                                                    
                                                                                                                                                                                                  
  필터 사이드바 (좌측):                                                                                                                                                                           
  - 가격대 범위 슬라이더                                                                                                                                                                          
  - 평점 필터 (별 4개 이상, 3개 이상 등 체크박스)                                                                                                                                                 
  - 응답 속도 (빠른응답 업체만 토글)                                                                                                                                                              
  - 인증 뱃지 필터 (사업자 인증, 프라임 업체 등)                                                                                                                                                  
  - 지역 선택 드롭다운                                                                                                                                                                            
                                                                                                                                                                                                  
  정렬 옵션 (필터 아래 또는 리스트 상단):                                                                                                                                                         
  - 추천순, 인기순, 최신순, 응답빠른순 탭 또는 드롭다운                                                                                                                                           
                                                                                                                                                                                                  
  전체적으로 깔끔하고 신뢰감 있는 의료 서비스 느낌. 메인 컬러는 청록색 또는 파란색 계열. 여백 충분히.

  2. 업체 리스트 페이지                                                                                                                                                                           
                                                                                                                                                                                                  
  의료 B2B 업체 목록 페이지를 디자인해줘.                                                                                                                                                         
                                                                                                                                                                                                  
  레이아웃:                                                                                                                                                                                       
  - 좌측에 필터 사이드바 (접을 수 있음)                                                                                                                                                           
  - 우측에 업체 카드 그리드 (한 줄에 3-4개, 반응형)                                                                                                                                               
                                                                                                                                                                                                  
  업체 카드 구성:                                                                                                                                                                                 
  - 상단: 대표 이미지 썸네일 (16:9 비율)                                                                                                                                                          
  - 이미지 우상단에 하트 아이콘 (찜하기)                                                                                                                                                          
  - 이미지 좌상단에 "빠른응답" 또는 "프라임" 뱃지 (해당 시)                                                                                                                                       
  - 하단 정보:                                                                                                                                                                                    
    - 업체명 (굵은 글씨)                                                                                                                                                                          
    - 한 줄 소개 (회색, 1줄 말줄임)                                                                                                                                                               
    - 별점 ★ 4.8 (노란색) + 후기 수 "(128)"                                                                                                                                                       
    - 가격대 "견적문의" 또는 "월 50만원~"                                                                                                                                                         
    - 카테고리 태그 칩 (작게)                                                                                                                                                                     
                                                                                                                                                                                                  
  카드 호버 시:                                                                                                                                                                                   
  - 그림자 깊어지고 살짝 위로 올라오는 애니메이션                                                                                                                                                 
  - "문의하기" 버튼 나타남                                                                                                                                                                        
                                                                                                                                                                                                  
  상단에 검색 결과 수 표시: "인테리어 업체 127개"                                                                                                                                                 
  무한 스크롤 또는 페이지네이션                                                                                                                                                                   
                                                                                                                                                                                                  
  모바일:                                                                                                                                                                                         
  - 카드 1열 또는 2열                                                                                                                                                                             
  - 필터는 하단 플로팅 버튼으로 모달 열기

  3. 업체 상세 페이지                                                                                                                                                                             
                                                                                                                                                                                                  
  의료 B2B 업체 상세 정보 페이지를 디자인해줘.                                                                                                                                                    
                                                                                                                                                                                                  
  히어로 섹션:                                                                                                                                                                                    
  - 대표 이미지 슬라이더 (좌우 화살표, 하단 도트 인디케이터)                                                                                                                                      
  - 이미지 우측 또는 하단에:                                                                                                                                                                      
    - 업체명 (큰 글씨)                                                                                                                                                                            
    - 한 줄 소개                                                                                                                                                                                  
    - 별점 ★ 4.8 + 후기 수                                                                                                                                                                        
    - 카테고리 브레드크럼 (인테리어 > 병원 인테리어)                                                                                                                                              
    - "찜하기" 버튼 (하트), "공유하기" 버튼                                                                                                                                                       
    - CTA 버튼: "문의하기" (강조색, 크게)                                                                                                                                                         
                                                                                                                                                                                                  
  탭 네비게이션 (스티키):                                                                                                                                                                         
  - 서비스 소개 | 가격 정보 | 포트폴리오 | 리뷰 | FAQ                                                                                                                                             
                                                                                                                                                                                                  
  서비스 소개 탭:                                                                                                                                                                                 
  - 상세 설명 (마크다운/리치텍스트)                                                                                                                                                               
  - 제공 서비스 목록 (체크 아이콘 + 텍스트)                                                                                                                                                       
  - 업체 정보 카드 (사업자명, 대표자, 연락처, 주소, 지도)                                                                                                                                         
                                                                                                                                                                                                  
  가격 정보 탭:                                                                                                                                                                                   
  - 가격표 테이블 또는 가격 카드                                                                                                                                                                  
  - "정확한 견적은 문의해주세요" 안내                                                                                                                                                             
                                                                                                                                                                                                  
  포트폴리오 탭:                                                                                                                                                                                  
  - 이미지 갤러리 그리드 (클릭 시 라이트박스)                                                                                                                                                     
  - 프로젝트별 카드 (이미지, 프로젝트명, 규모, 지역)                                                                                                                                              
                                                                                                                                                                                                  
  리뷰 탭:                                                                                                                                                                                        
  - 상단: 평균 별점 + 평점 분포 막대그래프                                                                                                                                                        
  - 정렬: 최신순, 평점높은순, 평점낮은순                                                                                                                                                          
  - 리뷰 카드:                                                                                                                                                                                    
    - 작성자 (닉네임, 프로필), 별점, 작성일                                                                                                                                                       
    - 계약 금액, 작업 기간 (선택)                                                                                                                                                                 
    - 리뷰 내용 텍스트                                                                                                                                                                            
    - 사진 첨부 시 썸네일                                                                                                                                                                         
                                                                                                                                                                                                  
  FAQ 탭:                                                                                                                                                                                         
  - 아코디언 형태 질문/답변 목록                                                                                                                                                                  
                                                                                                                                                                                                  
  하단 플로팅:                                                                                                                                                                                    
  - 모바일에서 "문의하기" 버튼 항상 하단에 고정

  4. 문의 작성 페이지                                                                                                                                                                             
                                                                                                                                                                                                  
  업체에 문의를 보내는 폼 페이지를 디자인해줘.                                                                                                                                                    
                                                                                                                                                                                                  
  상단:                                                                                                                                                                                           
  - 문의 대상 업체 정보 요약 카드 (썸네일, 업체명, 카테고리)                                                                                                                                      
                                                                                                                                                                                                  
  폼 섹션:                                                                                                                                                                                        
  - 서비스 선택 (해당 업체 서비스 중 선택, 드롭다운 또는 라디오)                                                                                                                                  
  - 이름 (텍스트 입력)                                                                                                                                                                            
  - 연락처 (전화번호 입력, 자동 하이픈)                                                                                                                                                           
  - 이메일 (이메일 입력)                                                                                                                                                                          
  - 소속 병원명 (텍스트 입력)                                                                                                                                                                     
  - 직책 (드롭다운: 원장, 부원장, 직원 등)                                                                                                                                                        
  - 선호 연락 채널 (라디오: 전화, 카카오톡, 이메일)                                                                                                                                               
  - 연락 가능 시간대 (체크박스: 오전, 오후, 저녁, 상관없음)                                                                                                                                       
  - 문의 내용 (텍스트에어리어, 최소 높이 150px)                                                                                                                                                   
  - 파일 첨부 (드래그앤드롭 영역, 복수 파일 가능)                                                                                                                                                 
                                                                                                                                                                                                  
  하단:                                                                                                                                                                                           
  - 개인정보 수집 동의 체크박스 + 약관 보기 링크                                                                                                                                                  
  - "문의 보내기" 버튼 (강조색, 넓게)                                                                                                                                                             
                                                                                                                                                                                                  
  성공 시:                                                                                                                                                                                        
  - 문의 완료 모달 또는 페이지                                                                                                                                                                    
  - "문의가 접수되었습니다. 업체에서 곧 연락드릴 예정입니다."                                                                                                                                     
  - "내 문의함 보기" 버튼                                                                                                                                                                         
                                                                                                                                                                                                  
  깔끔한 폼 UI, 필수 항목 표시(*), 실시간 유효성 검사

  5. 마이페이지 - 대시보드                                                                                                                                                                        
                                                                                                                                                                                                  
  한의사 회원의 마이페이지 대시보드를 디자인해줘.                                                                                                                                                 
                                                                                                                                                                                                  
  좌측 사이드바:                                                                                                                                                                                  
  - 프로필 영역 (프로필 사진, 이름, 병원명)                                                                                                                                                       
  - 메뉴 목록:                                                                                                                                                                                    
    - 대시보드 (현재, 강조)                                                                                                                                                                       
    - 내 문의함                                                                                                                                                                                   
    - 찜 리스트                                                                                                                                                                                   
    - 조회 목록                                                                                                                                                                                   
    - 리뷰 관리                                                                                                                                                                                   
    - 알림 설정                                                                                                                                                                                   
    - 계정 관리                                                                                                                                                                                   
                                                                                                                                                                                                  
  메인 콘텐츠:                                                                                                                                                                                    
  - 환영 메시지 "안녕하세요, 홍길동 원장님"                                                                                                                                                       
                                                                                                                                                                                                  
  요약 위젯 카드 (가로 3-4개):                                                                                                                                                                    
  - 진행 중 문의: 숫자 + 아이콘                                                                                                                                                                   
  - 대기 중 견적: 숫자 + 아이콘                                                                                                                                                                   
  - 새 알림: 숫자 + 빨간 뱃지                                                                                                                                                                     
  - 찜한 업체: 숫자                                                                                                                                                                               
                                                                                                                                                                                                  
  최근 활동 섹션:                                                                                                                                                                                 
  - 최근 문의 내역 리스트 (업체명, 상태 뱃지, 날짜) - 3~5개                                                                                                                                       
  - "전체 보기" 링크                                                                                                                                                                              
                                                                                                                                                                                                  
  추천 업체 섹션:                                                                                                                                                                                 
  - 관심 카테고리 기반 추천 업체 카드 슬라이더                                                                                                                                                    
                                                                                                                                                                                                  
  알림 섹션:                                                                                                                                                                                      
  - 최근 알림 3개 (아이콘, 내용, 시간)                                                                                                                                                            
  - "모두 보기" 링크                                                                                                                                                                              
                                                                                                                                                                                                  
  모바일:                                                                                                                                                                                         
  - 사이드바는 햄버거 메뉴로                                                                                                                                                                      
  - 위젯 카드 2x2 그리드

  6. 마이페이지 - 내 문의함                                                                                                                                                                       
                                                                                                                                                                                                  
  내가 보낸 문의 목록을 관리하는 페이지를 디자인해줘.                                                                                                                                             
                                                                                                                                                                                                  
  상단:                                                                                                                                                                                           
  - 페이지 제목 "내 문의함"                                                                                                                                                                       
  - 상태 필터 탭: 전체 | 대기중 | 진행중 | 완료 | 취소                                                                                                                                            
                                                                                                                                                                                                  
  문의 카드 리스트:                                                                                                                                                                               
  각 카드 구성:
