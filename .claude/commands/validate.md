# Validate: 코드 검증

## 목표

코드 변경 사항의 품질과 정확성을 검증합니다.

## 검증 단계

### Level 1: 정적 분석

```bash
cd app

# TypeScript 타입 체크
pnpm type-check

# ESLint 린트 체크
pnpm lint
```

**통과 기준**: 에러 0개

### Level 2: 단위 테스트

> **참고**: 테스트 프레임워크 미설정 상태. 이 단계는 건너뜁니다.

```bash
# 테스트 프레임워크 설정 후 사용
# pnpm test
# pnpm test -- path/to/test.ts
```

### Level 3: 빌드 검증

```bash
cd app

# 프로덕션 빌드
pnpm build
```

**통과 기준**: 빌드 성공, 에러/경고 없음

### Level 4: 데이터베이스 (해당 시)

```bash
cd app

# 마이그레이션 상태 확인
pnpm db:status

# 타입 동기화 확인
pnpm db:gen -- --local
```

**통과 기준**: 마이그레이션 동기화, 타입 일치
